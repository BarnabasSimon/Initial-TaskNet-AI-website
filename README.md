# TaskNest AI

TaskNest AI is a fictional AI-powered productivity SaaS website built as a front-end portfolio project.
The website presents a modern productivity platform that helps users plan their day, organize tasks, track focus sessions, and manage projects through an AI-assisted dashboard.

## Live Demo

Live site: https://initial-task-net-ai-website.vercel.app/

## Project Overview

This project was created to practice building a more advanced multi-page SaaS-style website using HTML, CSS, and JavaScript.

The goal was to make the site feel like a real software product, not just a simple landing page. It includes a full marketing homepage, feature page, pricing page, authentication pages, and a fictional app dashboard.

## Pages

* `index.html` — Homepage with hero section, product preview, features, testimonials, pricing preview, FAQ, and CTA
* `features.html` — Detailed feature overview for the fictional productivity app
* `pricing.html` — Pricing cards, billing toggle, comparison table, and FAQ
* `dashboard.html` — Static productivity dashboard demo with task cards, charts, calendar preview, and AI suggestions
* `login.html` — Fictional login page
* `signup.html` — Fictional signup page

## Features

* Responsive multi-page layout
* Dark futuristic SaaS design
* Glassmorphism-style cards
* Gradient backgrounds and glowing UI elements
* Fake AI productivity dashboard
* Pricing monthly/yearly toggle
* FAQ accordion
* Mobile navigation menu
* Dashboard task checkbox interactions
* Demo-only login and signup forms
